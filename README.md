# Chez Patron RougeO – Site Web Spécialisé en Freinage & Systèmes ABS

Site web professionnel pour un garage spécialisé dans la réparation et la vente de pièces de freinage et systèmes ABS à Treichville, Abidjan.

## Fonctionnalités
- Pages : Accueil, Services, Pièces de rechange, À propos, Contact, Blog.
- Technologies : Next.js, Tailwind CSS.
- Déploiement : Vercel (nom de domaine gratuit).

## Comment démarrer ?
```bash
npm install
npm run dev
